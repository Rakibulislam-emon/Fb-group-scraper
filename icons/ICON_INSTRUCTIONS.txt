ICON SETUP INSTRUCTIONS
========================

This extension requires three icon files to work properly:

Required Files:
- icon16.png (16x16 pixels) - Shows in browser toolbar
- icon48.png (48x48 pixels) - Shows in extensions page
- icon128.png (128x128 pixels) - Shows in Chrome Web Store

Quick Creation Methods:

METHOD 1: Use Online Tools (Easiest)
1. Visit https://favicon.io/favicon-generator/
2. Create a simple coffee cup ☕ or clock ⏰ icon
3. Download and extract the ZIP
4. Rename files to icon16.png, icon48.png, icon128.png
5. Copy them to this folder

METHOD 2: Use Canva
1. Go to https://www.canva.com/
2. Create a new design with custom dimensions
3. Add a coffee cup emoji or clock icon
4. Download as PNG in three sizes (16x16, 48x48, 128x128)
5. Save them in this folder

METHOD 3: Use ImageMagick (Command Line)
Run these commands in terminal:
  convert -size 16x16 xc:'#3b82f6' -pointsize 12 -fill white -gravity center -annotate +0+0 "⏰" icon16.png
  convert -size 48x48 xc:'#3b82f6' -pointsize 36 -fill white -gravity center -annotate +0+0 "⏰" icon48.png
  convert -size 128x128 xc:'#3b82f6' -pointsize 96 -fill white -gravity center -annotate +0+0 "⏰" icon128.png

METHOD 4: Temporary Placeholder (Testing Only)
For quick testing, create solid color squares:
  convert -size 16x16 xc:blue icon16.png
  convert -size 48x48 xc:blue icon48.png
  convert -size 128x128 xc:blue icon128.png

NOTE: The extension will NOT load without these icons!
After creating icons, reload the extension in chrome://extensions/
